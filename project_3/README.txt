This program scans through an input file(s) and records every word and its count in a binary tree.
It displays a table of these counts in all_counts.txt and prints the most common and least
common word to the terminal.

compile instructions: 
    gcc -Wall count_words.c binaryTree.c -o countWords

running the program:
    run the program by using the executable and a list of input file names
    for example: ./countWords input_data1.txt input_data2.txt
